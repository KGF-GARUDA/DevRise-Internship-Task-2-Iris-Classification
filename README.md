# 🌸 DevRise-Internship-Task-2-Iris-Classification

## 📌 Project Overview

This project was completed as part of the **DevRise AI & ML Internship – Task 2**.

The objective of this project is to build and evaluate multiple Machine Learning classification models on the Iris Flower dataset. The notebook demonstrates the complete Machine Learning workflow, including data preprocessing, exploratory data analysis (EDA), model training, evaluation, visualization, and performance comparison.

---

## 📂 Dataset

**Dataset:** Iris Flower Dataset

The dataset contains **150 samples** of iris flowers belonging to three species:

- Setosa
- Versicolor
- Virginica

### Features

- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

### Target

- Species
